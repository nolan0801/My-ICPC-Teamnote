# Titanic's ICPC Teamnote

> **Note:** This is our team's customized version of the **[justiceHui/icpc-teamnote](https://github.com/justiceHui/icpc-teamnote)**.

## What is this?

This is our team's code library (team note) for programming contests like ICPC.

It contains code for common algorithms and data structures that we can use during a competition.

## How to Build

This project uses LaTeX to create a PDF.

### 1. Overleaf (Recommended)

1.  Download this repository as a `.zip` file.
2.  On Overleaf, click **New Project**.
3.  Click **Upload Project** and choose the `.zip` file.
4.  It should compile automatically.

### 2. Build Locally

1.  You must have a LaTeX program installed (like TeX Live).
2.  Open your terminal and run:

    ```bash
    latexmk -pdf main.tex
    ```
